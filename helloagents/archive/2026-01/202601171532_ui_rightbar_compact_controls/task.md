# 轻量迭代：UI 右侧按钮收纳与合并（legacy /ui）

> 目标：不改业务逻辑，仅优化主界面右侧按钮的分类与交互，减少占宽与冗余。

- [√] 合并“开始监听/停止监听”为单一 toggle（图标/aria 文案随运行态变化）
- [√] 合并“退出/重启”为单一按钮：短按退出（确认），长按重启（可选确认）
- [√] 去除右侧工具栏容器的“肉眼可见外框”，并减少主内容区为右侧预留的无谓 margin
- [√] 同步更新知识库（wiki + CHANGELOG）并迁移方案包到 history
