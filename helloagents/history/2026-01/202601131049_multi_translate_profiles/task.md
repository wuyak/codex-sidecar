# 轻量迭代：多翻译 API Profiles（手动切换）

目标：在 sidecar UI 中支持保存多个翻译 HTTP 配置（Profiles），并可手动切换；同时保证下次启动可沿用已保存配置。

## 任务清单
- [√] 配置层：支持保存多个 HTTP Profiles（含选中项），兼容旧版单 URL 配置
- [√] 控制器：HTTP 翻译器根据选中 Profile 构建请求参数
- [√] UI：提供 Profile 下拉、添加/删除/切换，并与保存配置联动
- [√] 启动一致性：`./run.sh` 默认沿用已保存配置（不再被默认参数覆盖）
- [√] 文档：更新 sidecar 模块说明与使用方式
- [√] 验证：基础语法检查、启动配置读写与选择逻辑自检
